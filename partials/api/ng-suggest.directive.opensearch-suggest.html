opensearch-suggest
directive in module ng-suggest

Description

This directive can be used to provide a typeahead input field ... The current implementation of this directive will change

Use together with/requires ui.bootstrap.typeahead

  typeahead="item.label for item in suggest1($viewValue) | filter:$viewValue"
