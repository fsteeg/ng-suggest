ng-suggest

AngularJS module to use OpenSearch Suggestions and SeeAlso

ng-suggest is a module for AngularJS to facilitate the use of OpenSearch Suggestions and SeeAlso links in web application.

The latest release of ng-suggest is version 0.0.1-pre. Source code and issue tracker can be found at https://github.com/gbv/ng-suggest.

Download

The current release of ng-suggest is version 0.0.1-pre, including:

  • ng-suggest.min.js: minified version for production
  • ng-suggest.js: original source code for development
  • ng-suggest.css: stylesheet for the default templates
