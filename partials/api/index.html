ng-suggest

AngularJS module to provide Typeahead via OpenSearch Suggestions

ng-suggest is a module for AngularJS to facilitate the use of OpenSearch Suggestions and SeeAlso links in your web application. See ng-suggest for a detailed API documentation.

The latest release of ng-suggest is version 0.0.1-pre. Source code and issue tracker can be found at https://github.com/gbv/ng-suggest.
