OpenSearchSuggestions
service in module ng-suggest

Description

The OpenSearchSuggestions service can be used to query an OpenSearch server for search suggestions. The service is instanciated with an URL template that includes the character sequence {searchTerms}:

var url = "http://example.org/?q={searchTerms}";
var suggestService = new OpenSearchSuggestions(url);
suggestService.suggest(search).then( function(suggestions) {
     ...
});

The method suggestions returns a promise to get a suggestions object having the following form:

{
   query: "query string",
   values: [
       {
           label:       "completion",
           description: "optional description",
           url:         "optional URL"
       },
       ....
   ]
}
