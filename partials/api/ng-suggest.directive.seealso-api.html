seealso-api
directive in module ng-suggest

Description

This directive displaysy link suggestions queried via SeeAlso link server protocol.

Scope

The link suggestions are injected in the template's scope as query and links.

Source code

The most recent source code of this service is available at GitHub.

Usage

as attribute
<ANY seealso-api="{string}"
     jsonp="{string}"
     seealso-id="{string}"
     template-url="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
seealso-apistring

Base URL of SeeAlso server to query from

jsonpstring

enable JSONP (if service given as URL)

seealso-idstring

Identifier to query for

template-urlstring

Custom template to display result with
