SeeAlso
service in module ng-suggest

Description

The SeeAlso is a special kind of OpenSearchSuggestions service to query a SeeAlso server for link suggestions. A SeeAlso service is instanciated with the server's base URL as first argument:

var seealso = new SeeAlso("http://example.org/seealso");
seealso.suggest(search).then( function(links) { ... });

See seealso-api for easy usage of this service in a directive.
